Breadcrumb

Breadcrumbs show the user their current location within the page hierarchy and provide quick navigation back to parent pages. Essential on detail pages and settings sub-pages.

When to Use

Use Breadcrumb when
  • Detail pages (customer detail, user detail)
  • Settings sub-pages (Settings > Notifications)
  • Any page 2+ levels deep in the hierarchy
  • Users need to understand their current location
Don't use Breadcrumb when
  • Top-level pages — they have no parent to navigate to
  • Already shown in topbar breadcrumb for the page
  • Single-page apps with no real hierarchy

Anatomy

1
nav wrapper
<nav aria-label="Breadcrumb"> contains an <ol> list. Provides landmark navigation for screen readers.
2
Ancestor links
<a> elements. 13px color:#64748B. Hover: color:#6366F1. Underline on hover.
3
Separator
SVG chevron (/) or slash character. color:#CBD5E1. aria-hidden="true". Purely decorative.
4
Current page
Last item. Not a link. 13px fw-600 color:#0F172A. aria-current="page".

States

Breadcrumb items have two states: Link (ancestor pages — clickable, hover color change) and Current (last item — not clickable, fw-600, darker color, aria-current="page"). Separators are always static with no interactive state.

Behavior

TriggerBehavior
Ancestor link clickNavigates to href. Standard browser navigation.
Current item clickNo action — not a link. cursor:default.
Truncated "..." clickExpands to show all hidden intermediate items.

AI Implementation Guide

For AI coding assistants: Always use <nav aria-label="Breadcrumb"> > <ol> > <li> structure. Ancestor items are <a> tags. Last item is NOT a link — use <span aria-current="page">. Separators use aria-hidden="true". Separator: chevron-right SVG, color:#CBD5E1. Place breadcrumb below topbar, above page h1. Never show on top-level pages.

Design Decisions

Why chevron-right instead of slash as separator?
The chevron visually communicates directionality — it points forward, reinforcing the hierarchical flow from left (root) to right (current). A slash is ambiguous — it could represent a path separator, a date, or a ratio. The chevron is unambiguous.
