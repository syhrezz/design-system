Icons

Syhrezz uses inline SVG icons exclusively — from the Heroicons outline set. No icon fonts, no image files, no emoji. Every icon is rendered as an inline SVG with stroke="currentColor" so it inherits the text color of its parent.

Icon Rules

Never use emoji as icons. Never use icon fonts (Font Awesome, Lucide, etc.). Never use <img> tags for icons. Always use inline SVG with fill="none", viewBox="0 0 24 24", stroke="currentColor", and stroke-width="2".

AI Implementation Notes

For AI coding assistants: Always use inline SVG with these exact attributes: fill="none" viewBox="0 0 24 24" stroke="currentColor". All paths use stroke-linecap="round" stroke-linejoin="round" stroke-width="2". Size is set via inline style: style="width:Xpx;height:Xpx". Always add flex-shrink:0 when icon is inside a flex container. For icon-only buttons, always add aria-label.
