Grid & Layout

Syhrezz uses CSS Grid for all 2D layouts and Flexbox for 1D alignment. Every page follows the same shell structure: fixed sidebar + fixed topbar + scrollable main content. All grid values are named and documented here.

AI Implementation Notes

For AI coding assistants: Every page MUST have the sidebar (fixed, 268px) and topbar (fixed, 56px, left offset = sidebar width). Main content uses margin-left:268px; padding-top:56px. When sidebar collapses, both shift to 68px. Grid utilities use CSS Grid — never Flexbox for 2D layouts. The KPI row is ALWAYS grid-template-columns:repeat(4,1fr). Content max-width is ALWAYS 1400px with margin:0 auto and padding:28px 32px.
