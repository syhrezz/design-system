Border Radius

Syhrezz uses a fixed set of border radius values that define how rounded elements appear. These are not arbitrary — each token maps to a specific context. Never use values outside this scale.

Rules

Never use rounded-full on buttons. Buttons use 12px (md), 10px (Dark), 8px (sm), 7px (xs). Full pill buttons are not part of the Syhrezz design language. The --r-full token is reserved exclusively for badges, pills, avatars, and progress bar tracks.
