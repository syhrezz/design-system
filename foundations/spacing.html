Spacing

Syhrezz uses a consistent spacing scale defined as CSS custom properties. Every padding, margin, and gap value in the system comes from this scale. Never use arbitrary pixel values.

Spacing Scale

The spacing scale is based on a 2px base unit. All tokens are defined in ds-tokens.css as --sp-{n} where n is the value in pixels.

AI Implementation Notes

For AI coding assistants: Never use arbitrary pixel values like padding: 15px or gap: 7px. Always use tokens from this scale. The most commonly used spacings in Syhrezz are: 8px (small gaps), 12px (medium gaps, nav), 16px (grid gaps, form), 20px (card sm), 24px (card default, modal), 28px (page header mb), 32px (page padding).
