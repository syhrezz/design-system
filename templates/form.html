Form Page Template

The form page template is used for standalone create/edit pages where the form is too complex for a modal. It wraps the form in a card with a clear page header, sectioned fields, and a sticky footer with actions.

Form Sections

Long forms use section headers to group related fields. Section headers are 14px fw-700 with a bottom border, margin-bottom:20px. Each section is separated by margin-top:28px.

AI Implementation Notes

For AI coding assistants: Form page uses display:grid; grid-template-columns:2fr 1fr; gap:16px for main layout — form on left, summary/preview on right. Section headers use 14px fw-700 with border-bottom:1px solid var(--border) and margin-bottom:18px. Two-column field pairs inside sections use CSS Grid grid-template-columns:1fr 1fr; gap:14px. Cancel always navigates back. Save shows loading spinner then success toast on completion.
