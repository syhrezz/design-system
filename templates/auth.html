AI Implementation Notes

For AI coding assistants: Auth pages do NOT use the sidebar/topbar shell. They are full-viewport layouts. The split panel uses display:flex; min-height:100vh. Left panel is flex:1 with the dark gradient background. Right panel is width:380-420px; flex-shrink:0; background:#fff. The submit button is always full width (width:100%). Each role has its own login route. Feature icons use inline SVG in a small rounded-square container (24px, bg:rgba(255,255,255,.08)).
