Working Reference

The sample-dashboard.html file included with this design system is the canonical working implementation for this template — fully functional HTML with all components composed correctly, including animated KPI cards, Chart.js charts, follow-up list, activity feed, data table, todo list, and toast notifications.

AI Implementation Notes

For AI coding assistants: Always include a toast container at the bottom of the page that auto-fires a welcome/info toast after 700ms. KPI cards use .fade-up .fu-1 through .fu-4 stagger animation. Charts require Chart.js CDN and must have responsive:true, maintainAspectRatio:false in options. The follow-up list left border colors are: overdue=#F05252, today=#E3A008, upcoming=#0E9F6E. Todo priority dots: red=#F05252 (high), amber=#E3A008 (medium), slate=#CBD5E1 (low).
