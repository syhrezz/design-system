Changelog

A record of all significant changes, additions, and fixes to the Syhrezz Design System. Follow semantic versioning: MAJOR.MINOR.PATCH.

v1.0.0 — July 24, 2026

Major release. Complete design system documentation platform. 50 pages across foundations, components, patterns, templates, and resources.

Added

  • New documentation platform at design-system/v2/ with searchable sidebar, keyboard shortcuts, prev/next navigation, and on-page TOC
  • Foundations (10 pages): Colors, Typography, Spacing, Border Radius, Shadows, Motion, Icons, Grid & Layout, Breakpoints, Design Tokens
  • Components (20 pages): Button, Badge, Input, Select, Textarea, Checkbox, Card, Table, Modal, Dropdown, Tabs, Toast, Alert, Avatar, Progress, Tooltip, Sidebar, Topbar, Pagination, Empty State — all with live playgrounds, do/don'ts, and AI implementation notes
  • Patterns (8 pages): Dashboard Layout, CRUD Table, Form Layout, Filter & Search, Loading States, Error States, Empty States, Approval Flow
  • Templates (5 pages): Dashboard, List Page, Detail Page, Form Page, Auth Page — all with wireframes and composition guides
  • Resources (5 pages): AI Implementation Guide, Accessibility, Component API, Writing Guidelines, Changelog
  • Shared infrastructure: ds-tokens.css (single source of truth), ds-shell.css (documentation chrome), ds-shell.js (search, nav, playgrounds, copy buttons, prev/next, TOC)
  • Dynamic path-prefix resolution — all nav links work correctly from any page depth
  • IntersectionObserver-based active nav highlighting for anchor-based items
  • Interactive playgrounds for Button, Badge, Input, and Progress components
  • Sidebar scroll fix: height:100vh + overflow-x:hidden

Changed

  • Replaced all emoji icons with inline SVG (Heroicons outline) across all design system HTML files
  • CSS checkmark in .todo-check::after changed to Unicode escape \2713
  • Hero heading "Build faster. Stay consistent." fixed to white color on dark gradient background

Fixed

  • Sidebar nav active state — triple-active bug when all Getting Started items shared the same base URL
  • Content right-alignment — removed display:block override and inner wrapper div from index.html
  • Quick Start and Design Principles anchor links — now smooth-scroll correctly from sidebar
  • Sidebar scroll — min-height:100vh → height:100vh, overflow:hidden → overflow-x:hidden

v1.0.0 — July 23, 2026

Initial design system extraction from Syhrezz prototype.

Added

  • design-system/index.html — single-file design system reference (1,830 lines, 117 KB)
  • design-system/addons.html — extended components from live prototype (1,113 lines, 70 KB)
  • sample-dashboard.html — full working Dashboard implementation (856 lines, 50 KB)
  • design-system/components.jsx — React component library (1,051 lines, 45 KB)
  • All CSS custom properties extracted as design tokens
  • KPI cards, sidebar variants, topbar, modal, table, badges, activity feed, follow-up list, Location badges, and more documented with live previews
